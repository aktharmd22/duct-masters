# duct-masters

HVAC Duct Manufacturing & Sheet Metal Fabrication website for Duct Masters (Musaffah, Abu Dhabi).

Built with **Astro 5** + **Tailwind CSS v4**. Static site (67+ pages) covering services, service areas, industries, and a blog — with a modern, animated UI.

## Commands

| Command | Action |
|---|---|
| `npm install` | Install dependencies |
| `npm run dev` | Start local dev server at `localhost:4321` |
| `npm run build` | Build the static site to `./dist` |
| `npm run preview` | Preview the production build locally |
